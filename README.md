# odin-landing-page
Landing Page Template 

This project is a beginner landing page layout made using flex-box. 

To complete this project I used HTML and CSS to practice: 

- Adding images to a webpage 
- Editing the sizes of images 
- Arrange items on a page using flex box display

Things I learned while completing  this project: 

- The key idea of flex-box is items in containers 
- Setting set dimensions on items can be helpful to make them appear a certain size 
- The importance of having clear git commits!
- Its easier to write all the css (even if its a little messy first), then clean it up and group common styles together
